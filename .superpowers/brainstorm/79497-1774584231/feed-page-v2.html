What each tab shows:

Scores

Live + recently finished games. Card grid with team codes, logos, full names, scores, game clock. Real-time CDC updates. This is shown above.

Schedule

Upcoming games grouped by date. Compact rows: team codes + logos + start time. "Today", "Tomorrow", then date headers. Same data, filtered to state='pre'.

Standings

League table per league. Rank, team, W-L, win%, points for/against. Grouped by conference/division where applicable. Fetched daily from api-sports.io.
